# 🧠 心理健康指导师考试模拟系统

## 🚀 快速启动

```bash
cd /root/.openclaw/workspace/exam-system
streamlit run app.py --server.port 8501
```

访问：http://localhost:8501

## 📋 功能特性

### MVP 版本功能
- ✅ **Word 题库导入** - 支持 .docx 格式，自动解析单选题和案例分析题
- ✅ **答题模式** - 逐题展示，选择题即时判分
- ✅ **问答题关键词匹配** - 自动匹配心理学专业术语，60% 匹配度算正确
- ✅ **错题集** - 自动收集错题，支持重复练习
- ✅ **统计分析** - 正确率、按题型统计

### 题库格式要求

#### 单选题格式
```
题目：人生第一发展加速期是（）。
选项：A.从出生到幼儿期 B.童年期 C.青春发育期 D.成年期
答案：A
分析：个体心理发展具有不平衡性...
```

#### 案例分析题格式
```
案例：22 岁大学毕业生，求职不顺，焦虑、烦躁...
问题
1. 求助者主要症状是什么？
2. 初步诊断及依据？
答案
症状：焦虑、烦躁、后悔...
诊断：一般心理问题...
```

## 🔧 配置说明

### 问答题评分规则
- 从答案中提取心理学专业关键词
- 用户答案与关键词匹配
- 匹配度 ≥ 60% 判定为正确

### 关键词库
内置心理学常见术语：
- 症状类：焦虑、抑郁、恐惧、强迫...
- 诊断类：一般心理问题、严重心理问题、神经症...
- 理论类：ABC 理论、合理情绪疗法、认知行为...
- 疗法类：精神分析、人本主义、行为主义...

## 📝 使用流程

1. **导入题库** - 上传 Word 文档（.docx）
2. **开始答题** - 选择答题模式，逐题作答
3. **查看答案解析** - 提交后立即显示
4. **查看错题集** - 复习薄弱环节
5. **查看统计** - 了解学习进度
