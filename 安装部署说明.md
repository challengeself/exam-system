# 🧠 心理健康指导师考试模拟系统 - 安装部署说明

## 📦 一、获取程序

程序已打包为：`exam-system.tar.gz`（约 15KB）

---

## 🛠️ 二、环境要求

- **操作系统**：Windows / macOS / Linux
- **Python 版本**：Python 3.8 或更高
- **内存**：至少 512MB 可用内存

### 检查 Python 版本

```bash
python --version
# 或
python3 --version
```

如果未安装 Python，请访问：https://www.python.org/downloads/

---

## 📥 三、安装步骤

### 步骤 1：解压程序包

```bash
# Linux / macOS
tar -xzvf exam-system.tar.gz

# Windows（使用 7-Zip 或 WinRAR 解压）
```

解压后得到 `exam-system` 文件夹。

### 步骤 2：进入项目目录

```bash
cd exam-system
```

### 步骤 3：安装依赖

```bash
# Windows
pip install -r requirements.txt

# macOS / Linux
pip3 install -r requirements.txt
```

如果提示权限问题，添加 `--user` 参数：

```bash
pip install -r requirements.txt --user
```

### 步骤 4：启动程序

```bash
# Windows
streamlit run app.py

# macOS / Linux
streamlit run app.py --server.port 8501
```

### 步骤 5：打开浏览器

访问地址：**http://localhost:8501**

---

## 🚀 四、快速启动脚本

### Windows 用户

创建 `start.bat` 文件，内容：

```batch
@echo off
echo 启动心理健康指导师考试模拟系统...
streamlit run app.py
pause
```

双击运行即可。

### macOS / Linux 用户

已提供 `start.sh` 脚本：

```bash
chmod +x start.sh
./start.sh
```

---

## 📚 五、导入题库

1. 打开浏览器访问 http://localhost:8501
2. 点击左侧菜单「📚 题库导入」
3. 上传 Word 题库文件（.docx 格式）
4. 系统自动解析题目

### 题库格式要求

#### 单选题格式
```
题目：人生第一发展加速期是（）。
选项：A.从出生到幼儿期 B.童年期 C.青春发育期 D.成年期
答案：A
分析：个体心理发展具有不平衡性...
```

#### 案例分析题格式
```
案例：22 岁大学毕业生，求职不顺，焦虑、烦躁...
问题
1. 求助者主要症状是什么？
2. 初步诊断及依据？
答案
症状：焦虑、烦躁...
诊断：一般心理问题...
```

## 💾 六、数据存储

所有数据保存在 `storage/` 文件夹：

- `questions.json` - 题库缓存
- `wrong_notes.json` - 错题集
- `history.json` - 答题历史

**注意**：更换电脑时请复制整个 `storage` 文件夹以保留学习进度。

---

## 🔧 七、常见问题

### 1. 提示 "streamlit: command not found"

解决方法：

```bash
pip install streamlit
```

### 2. 端口被占用

如果 8501 端口被占用，可指定其他端口：

```bash
streamlit run app.py --server.port 8502
```

### 3. 无法打开浏览器

手动访问：http://localhost:8501

或在启动时禁用自动打开浏览器：

```bash
streamlit run app.py --server.headless true
```

### 4. Word 文档解析失败

检查文档格式是否符合要求：
- 使用「题目：」「选项：」「答案：」「分析：」等标准格式
- 避免使用复杂表格或图片
- 确保是 .docx 格式（不是 .doc）

---

## 📞 八、技术支持

如有问题，请检查：

1. Python 版本是否 >= 3.8
2. 依赖是否全部安装成功
3. 防火墙是否阻止 8501 端口

## 🎯 九、使用流程

```
1. 启动程序 → streamlit run app.py
2. 打开浏览器 → http://localhost:8501
3. 导入题库 → 上传 Word 文档
4. 开始答题 → 选择题/问答题
5. 查看答案解析 → 即时反馈

7. 查看统计 → 学习进度分析
```
